Projet : MystikJumping3D
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------
                                                             !!! Bienvenue dans MystikJumping3D !!!
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Vous incanez un aventurier assoiffé de sensations et de parcours toujours plus risqués !

A vous d'enchaîner les sauts afin de parcourir le fameux "Sentier Perdu des Iles Volantes"...

le but du projet est de faire un jeu style parcours saut d'obstacle avec un joueur evoluant dans un decor
il y a aussi un menu principal dans une autre scene que celle de jeu avec un bouton pour lancer le jeu et un bouton pour quitter
De meme il y a aussi un menu pause en appuyant sur echap avec un bouton pour reprendre le jeu, un pour retourner au menu principal 
et un pour quitter.
           ------------------------
          |  COMMANDES :           |
          | -----------------------|
          |  w,a,s,d : se deplacer |       Note : Utiliser un clavier qwerty.
          |  shift : accelerer     |
          |  espace : sauter       |
          |  echap : pause (menu)  |
           ------------------------

Features :
 - Menu permettant de mettre pause, fermer le jeu et reprendre la partie.
 - Particle systems,
